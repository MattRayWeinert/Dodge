    Glowworm

    As my first Computer Science project, I was instructed to develop a program that prints out a worm and a ledge. Based off of the commands passed into the function, the worm would perform actions. Arrays were restricted from use, therefore I made a creative way to store data with brute force algorithms.

    Key Features

    • Programed in C using an open source text editor and a linux command line compiler
    • Designed to pass commands into algorithms without the use arrays
    • Constructed algorithms to implement the inputted commands onto the glowworm

    LonelyPartyArray

    This projcet required knowledge in memory allocation and memory management. The requirement was to utilize the minimal amount of memory required to store as much data as needed. The data was stored into an allocated struct, which held data which also inlucded allocated arrays. The user had multiple optons to modify the data with, such as adding data, removing, resetting all the data, destroy specific data and a few more options. If all the data in an allocated memory location was destroyed, the memory would be free'd, and so forth.

    Key Features

    • Programed in C using an open source text editor and a linux command line compiler
    • Constructed to manage memory in the interest of memory conservation
    • Created with functions to handle all the tasks prompted by the user
